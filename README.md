# Customer Churn Prediction

##  Project Overview
This project predicts customer churn using Machine Learning.

##  Tools Used
- Python (Pandas, NumPy, Scikit-learn)
- SQL
- Power BI

## Features
- Data preprocessing
- Random Forest model
- Churn prediction
- Data visualization

##  Result
Achieved ~83% accuracy in predicting customer churn.
