# Customer Churn Prediction

##  Project Overview
This project focuses on analyzing customer behavior to predict customer churn using machine learning techniques. Customer churn refers to the situation where customers stop using a company’s services, which directly impacts revenue and business growth. The main objective of this project is to identify customers who are likely to leave and help businesses take proactive measures to retain them.

The project uses a dataset from LoyaltyVision Analytics, which includes customer details such as payment methods, tenure, revenue, and service interactions. The data is first cleaned and preprocessed using Python libraries like Pandas and NumPy. Categorical variables are encoded, and missing values are handled to prepare the dataset for analysis.

A Random Forest Machine Learning model is implemented using Scikit-learn to predict whether a customer will churn or not. The model is trained and tested on the dataset, achieving an accuracy of around 80–85%, making it reliable for business use.

To enhance understanding, data visualization techniques are applied using Matplotlib, Seaborn, and Power BI. These visualizations help identify key factors influencing churn, such as payment method, customer tenure, and service usage patterns.

The results of the project provide valuable insights that can help businesses:

Identify high-risk customers
Improve customer satisfaction
Design targeted retention strategies
Increase long-term profitability

Overall, this project demonstrates how data analytics, machine learning, and business intelligence tools can be combined to solve real-world business problems and support data-driven decision-making.
